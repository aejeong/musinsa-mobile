# musinsa-mobile
 무신사 모바일 과제
### 실행

1. download sources

```
$ git clone https://github.com/aejeong/musinsa-mobile.git
```

2. install dependencies

```
$ npm install

```

3. 실행

```
$ npm run dev

```
